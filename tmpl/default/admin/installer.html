Для утсновки расширения вам необходимо выбрать пункт "Установка".
Для обновления системных параметров расширения (алиас, привязка входных точек к функциям и др.) вам необходимо выбрать пункт "Обновление".

В данной версии скрипта установки нет возможности устанавливать расширения из галереи расширений на сайте проекта.
Все действия выполняются без проверки совместимости текущей версии CMS с утанавливаемыми расширениями, рекомендуется узнавать о работоспособности до их установки.
